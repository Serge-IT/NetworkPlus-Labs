# Project 1: Set Up a Windows Virtual Machine Using VirtualBox

## Objective
Set up and configure a Windows virtual machine using Oracle VirtualBox while learning virtualization concepts and resource allocation.

## Project Description
In this project, I installed Oracle VirtualBox, verified hardware virtualization enabled in UEFI firmware, created a Windows virtual machine, and configured networking connectivity inside the guest operating system.

## Technologies and Tools Used
- Oracle VirtualBox
- Windows 11
- Windows ISO Image
- UEFI Firmware Settings

## Skills Demonstrated
- Virtual machine deployment
- Hypervisor installation and configuration
- Hardware-assisted virtualization (HAV)
- Operating system installation
- Virtual networking
- Resource allocation
- Troubleshooting and research skills

## Tasks Completed
1. Verified hardware-assisted virtualization enabled in UEFI
2. Installed Oracle VirtualBox
3. Created a new Windows virtual machine
4. Attached a Windows ISO image
5. Configured guest OS username and password
6. Allocated memory and storage resources
7. Installed Windows inside the VM
8. Verified Internet connectivity
9. Shut down the VM properly
10. Removed unattended installation attachment

## VM Configuration
- Hypervisor: Oracle VirtualBox
- Guest OS: Windows 11
- Virtual Disk: Default VirtualBox allocation
- Network Mode: NAT (default)

## Troubleshooting Performed
- Verified virtualization support in UEFI
- Confirmed Internet connectivity inside VM

- Researched virtualization settings when necessary

## What I Learned
This project introduced virtualization concepts and demonstrated how hypervisors allow multiple operating systems to run on one physical machine. I also learned how to configure VM resources and troubleshoot installation issues.
