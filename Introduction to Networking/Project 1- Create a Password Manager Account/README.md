# Project 1: Create a Password Manager Account

## Objective
Learn how to securely manage passwords using a password manager and explore account security features such as two-step login and vault protection.

## Project Description
In this project, I created a Bitwarden password manager account to securely store credentials for networking and IT-related labs completed during my CompTIA Network+ studies.

## Technologies and Tools Used
- Bitwarden
- Web Browser
- Internet Connection

## Skills Demonstrated
- Password security best practices
- Secure credential management
- Two-factor authentication awareness
- Account security configuration
- Cybersecurity fundamentals

## Steps Completed
1. Created a Bitwarden account
2. Configured a secure master password
3. Verified email address
4. Explored the Bitwarden vault
5. Created and tested a vault entry
6. Reviewed auto-fill and security settings
7. Logged out securely after testing

## Security Concepts Learned
- Password managers help reduce password reuse
- Strong master passwords improve account security
- Two-step login adds additional protection
- Secure storage of credentials is essential in cybersecurity

## Reflection
This project introduced the importance of password management and account security in IT environments. I learned how password managers can securely store credentials while improving convenience and security.
